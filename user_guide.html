RETRO MOTION DETECTOR

USER GUIDE

1. GETTING STARTED

The Retro Motion Detector is a web application that uses your device's camera to detect motion in real-time. Follow these steps to begin:

  1. Open index.html in a modern web browser
  2. Click the START button
  3. Allow camera access when prompted by your browser
  4. The application will immediately begin monitoring for motion
TIP: For best results, position your camera to view the area you want to monitor and ensure adequate lighting.

2. UNDERSTANDING THE CONTROLS

Control Function
START Button Begins the motion detection process and activates the camera
STOP Button Stops motion detection and deactivates the camera
SENSITIVITY Slider Adjusts how sensitive the detector is to pixel changes (10-100)
MIN PIXELS Slider Sets the minimum number of changed pixels required to trigger detection (500-10000)
SOUND Toggle Enables or disables the audio alert when motion is detected

3. READING THE INDICATORS

Indicator Meaning
STATUS Text NO MOTION (green): No movement detected
MOTION DETECTED (red): Movement detected
MOTION LEVEL Bar Visual representation of the amount of motion detected - longer bar indicates more motion
RESOLUTION Current video resolution being processed
FPS Frames Per Second - indicates processing performance

4. ADJUSTING SENSITIVITY FOR OPTIMAL DETECTION

Finding the right sensitivity settings is crucial for effective motion detection. Here's how to adjust the settings for different scenarios:

For General Use:

  • SENSITIVITY: 30-50
  • MIN PIXELS: 1500-3000

For Detecting Small Movements:

  • SENSITIVITY: 60-80
  • MIN PIXELS: 500-1000

For Reducing False Alarms:

  • SENSITIVITY: 20-30
  • MIN PIXELS: 3000-5000
TIP: If you're getting too many false detections, increase the MIN PIXELS value. If the application is missing movements, increase the SENSITIVITY value.

5. TROUBLESHOOTING

Issue Solution
Camera not working Check browser permissions and allow camera access
Low performance (low FPS) Lower the resolution in your browser's camera settings
No sound alerts Ensure the SOUND toggle is checked and your system volume is not muted
Too many false detections Increase MIN PIXELS and/or decrease SENSITIVITY
Missing motion events Increase SENSITIVITY and/or decrease MIN PIXELS
